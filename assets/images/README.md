# Carpeta de Imágenes

Coloca aquí las imágenes para cada vía verde.

```

├── hero/                    # Imágenes principales de cabecera
│   ├── vv1-campo-cartagena.jpg
│   ├── vv2-mazarron.jpg

│   ├── vv1-campo-cartagena-thumb.jpg
│   ├── vv2-mazarron-thumb.jpg
│   └── ...
│
├── gallery/                 # Galería de fotos adicionales
│   ├── vv1/
│   │   ├── 01.jpg
│   │   ├── 02.jpg
│   │   └── ...
│   └── ...

    ├── marker-operativa.png
    ├── marker-proximamente.png
    └── logo-vv-murcia.png
```

## Medidas recomendadas

| Tipo | Dimensiones | Formato | Uso |
|------|-------------|---------|-----|
| **Hero** | 1200 x 600 px | JPG/WebP | Cabecera del modal de detalle |
| **Thumbnail** | 400 x 250 px | JPG/WebP | Tarjetas en el listado |
| **Gallery** | 800 x 600 px | JPG/WebP | Galería de fotos |
| **Logo** | 512 x 512 px | PNG (transparente) | Logo principal |
| **Marker** | 64 x 64 px | PNG (transparente) | Marcadores del mapa |
| **OG Image** | 1200 x 630 px | JPG | Redes sociales (og:image) |

## Optimización

- Comprime las imágenes antes de subirlas (TinyPNG, Squoosh)
- Usa WebP cuando sea posible para mejor rendimiento
- Mantén las imágenes hero por debajo de 200KB
- Mantén los thumbnails por debajo de 50KB
